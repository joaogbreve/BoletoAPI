código de barras -> cpf/cnpj -> nome social

API recebe código de barras, cpf/cnpj e nome social para pagamento do boleto:

Código de barras presente no banco -> autoriza
Código não está presente -> verifica cpf/cnpj
                            presente -> erro de código de barras/vencido/inválido
                            ausente -> compara o cpf/cnpj com o nome social apresentado
                                    ausente -> beneficiário não cadastrado no sistema
                                    presente -> boleto suspeito, pagamento não recomendado
